Use v11 account groups feature / account_group

This modules brings to version 10 the account group feature from v11.

It also includes a level field for indicating the level depth of the account group. </p>

Traduzione non disponibile</p>

Usage / Utilizzo

For assigning groups to accounts:

  • Invoicing > Adviser > Chart of Accounts*.

Edit one account and set "Group" field.
